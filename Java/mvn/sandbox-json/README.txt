This project (sandbox-json) has example of 3 different JSON libraries:

  * src/main/.../DemoGson - google-gson
      http://code.google.com/p/google-gson/
      
  * src/main/.../DemoJsonLib - Json-lib
      http://json-lib.sourceforge.net/
      
  * src/test/.../TestEncoding - json-simple
      http://code.google.com/p/json-simple/
      
 The POM contains the <dependency> declaration for each of the projects.
